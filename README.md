
### 脑机交互课程预习资料

1. BCI 相关资料：参阅`docs/lotte_EEGSignalProcessing.pdf`，主要是让大家对BCI有一个基本的了解，快速过一遍即可；

3. 机器学习相关：参阅`doc/机器学习预习指导.txt`，其中标注了参考书籍以及参考资料中的重点章节，上述内容基本了解即可；
